# نظام بوابة كلية القاسمي التقنية 
هذا المشروع عبارة عن بوابة ويب تفاعلية لطلاب كلية القاسمي، تتضمن نظام دردشة ذكي (Chatbot) مبني باستخدام Flask وDocker.
## المميزات 
- **مساعد تقني ذكي**: يرد على استفسارات الطلاب بخصوص (Moodle، البريد الإلكتروني، كلمات المرور).
- **دعم لغات متعددة**: يفهم الكلمات المفتاحية بالعربية، العبرية، والإنجليزية.
- **تقنيات حديثة**: يعمل باستخدام Docker لضمان سهولة التشغيل في أي بيئة.

## التقنيات المستخدمة 
- **Backend**: Python (Flask)
- **Frontend**: HTML/CSS/JS
- **DevOps**: Docker

## كيفية التشغيل 
1. قم بتشغيل خادم الـ API:
   `python3 app.py`
2. قم بتشغيل واجهة المستخدم عبر Docker:
   `docker run -d -p 8080:80 --name my-portal it-portal`
